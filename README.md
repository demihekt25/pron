# Сайт Школы Кройки и Шитья

Гибридный сайт школы кройки и шитья с главной страницей-лендингом и отдельными страницами для курсов, индивидуальных занятий и коворкинга.

## Структура проекта

```
site/
├── index.html              # Главная страница-лендинг
├── about.html              # О школе
├── reviews.html            # Отзывы
├── contacts.html           # Контакты
├── inlesson.html           # Индивидуальные занятия
├── covorking.html          # Коворкинг
├── courses/
│   ├── index.html          # Каталог курсов
│   ├── belye.html          # Курс: Нижнее бельё
│   ├── trikotag.html       # Курс: Трикотаж
│   ├── domodegda.html      # Курс: Домашняя одежда
│   ├── platye.html         # Курс: Платье
│   ├── ubka.html           # Курс: Юбка
│   ├── detodegda.html      # Курс: Детская одежда
│   ├── bruki.html          # Курс: Брюки
│   ├── gaket.html          # Курс: Жакет
│   └── kupalnik.html       # Курс: Купальник
├── admin/
│   ├── index.html          # Панель администратора
│   └── login.html          # Вход в админ-панель
├── css/
│   ├── variables.css       # CSS переменные из design.json
│   └── main.css            # Основные стили
├── js/
│   └── main.js             # JavaScript функционал
├── design.json             # Дизайн-система
└── README.md               # Этот файл
```

## Технологии

- **HTML5** - семантическая разметка
- **CSS3** - стили на основе design.json
- **JavaScript** - интерактивность (формы, аккордеоны, фильтры)
- **Адаптивный дизайн** - mobile-first подход
- **SEO оптимизация** - Schema.org разметка

## Особенности

### Дизайн
- Все стили основаны на `design.json`
- Адаптивность для desktop (1440px), tablet (768px), mobile (375px)
- Использование цветовой палитры, типографики и компонентов из дизайн-системы

### Функционал
- **Формы заявок** с валидацией на клиенте
- **Аккордеоны** для FAQ и программ курсов
- **Фильтры расписания** по уровню, формату и дню недели
- **Ленивая загрузка изображений** (lazy loading)
- **Фиксированная кнопка CTA** на мобильных устройствах

### SEO
- ЧПУ-URL для всех страниц
- Один H1 на страницу
- Meta-теги (title, description)
- Schema.org разметка (Organization, Course, Review, LocalBusiness)

### CMS
- Базовая панель администратора
- Управление курсами
- Управление расписанием
- Просмотр заявок
- Редактирование контента

## Установка и запуск

1. Скопируйте все файлы на веб-сервер
2. Убедитесь, что сервер поддерживает статические файлы
3. Для работы форм требуется настроить бэкенд (API endpoints)

## Настройка форм

Формы настроены на отправку данных на `/api/submit-form`. Для работы необходимо:

1. Создать API endpoint для обработки форм
2. Настроить отправку уведомлений в Telegram (опционально)
3. Настроить отправку email уведомлений (опционально)

Пример настройки в `js/main.js`:
```javascript
fetch('/api/submit-form', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
    },
    body: JSON.stringify(data)
})
```
